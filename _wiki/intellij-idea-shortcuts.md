## intellij-idea-cheat-sheet
Some useful shortcuts for IntelliJ IDEA

### Find And Replace
| Shortcuts | Description |
| ------- | ------- |
| ctrl+F | Find |
| ctrl+R | Replace |
| ctrl+L | Find next |
| shift+ctrl+L | Find previous |
| shift+ctrl+F | Find in path |
| shift+ctrl+R | Replace in path |
| alt+F7 | Find usages |
